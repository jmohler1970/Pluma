Requirements

  • SQL Server 2008 and above. SQL 2012 recommended
  • ColdFusion 9.01 and above. Hot fix 4 recommended

Installation:

1. Build the SQLServer DB

This can be found in: _install/0 - drop all objects.sql (Optional) _install/1 - functions.sql _install/2 - tables.sql _install/3 - view.sql _install/4 - storedprocedures.sql _install/5 - triggers.sql _install/6 - indexes.sql _install/9 - load data.sql (Recommended)

2. Go to the login page

The installation tool can reached by: ../index.cfm/login
