Requirements

  • SQL Server 2008 and above. SQL 2012 recommended
  • ColdFusion 9.01 and above. Hot fix 4 recommended

Installation:

1. Build the SQLServer DB

This can be found in: install/1 - functions.sql install/2 - tables.sql install/3 - view.sql install/4 - storedprocedures.sql install/5 - triggers.sql install/6 - indexes.sql

2. Go to the install tool

The installation tool can reached by: yourhost.cfm/yourhost/index.cfm/install
